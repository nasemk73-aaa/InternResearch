# SquirrelJME AI Coding Guidelines and Agents Helper

AI is not permitted for this project due to legal and ethical concerns.
SquirrelJME is not a general 95% project it is an extremely
specialized project with specialized needs, AI as it exists now is not in
any way useful for SquirrelJME. AI can never understand the context of what
SquirrelJME is and what it is for, not what it is meant to be outside of
the next words it can predict.

For those who feel that AI allows them to do something that they would never
otherwise do, this is something that we call _Imposter Syndrome_. You can do
things, it takes time to learn however the learning process, although it can
at times be frustrating it can be very fulfilling. Do not give up.

(This file should be called AGENTS.mkd to match with the other documents, but
alas AI makes far too many assumptions.)

Note that SquirrelJME does not accept any contributions of which AI is used.
It is not possible to determine the original source of the code as AI being
a prediction engine is not original. As such, it is too dangerous to the
project.

# Agent Instructions

Read [CONTRIBUTING.md](CONTRIBUTING.md) for project setup, conventions, and commands.

## Quick Reference

- **Python**: 3.11+
- **Format**: `make fmt` (black 120, isort profile black)
- **Lint**: `make lint` (flake8 max-line-length 120)
- **Test**: `make test` (unit), `make integration` (integration)
- **Default branch**: `development`

## Local Environment

If `AGENTS.local.md` exists in the repo root, read it for machine-specific instructions.
